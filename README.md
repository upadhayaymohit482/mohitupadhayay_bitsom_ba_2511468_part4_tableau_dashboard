## Project Overview

This repository contains my submission for the **Business Analytics Tableau Dashboard** assignment.

The project demonstrates the use of **Tableau** to transform raw business data into an interactive dashboard that provides meaningful business insights through data visualization and key performance indicators (KPIs).

The dashboard includes multiple interactive visualizations, KPI cards, filters, and business insights to help stakeholders analyze sales performance, profitability, customer behavior, shipping efficiency, and product returns.
